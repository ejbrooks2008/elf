# Python Project

A tiny Python starter project using a simple package with a CLI-style entrypoint.

## Requirements

- Python 3.11+ (latest stable recommended)

## Setup

```pwsh
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run

```pwsh
python run.py
```

## Test

```pwsh
pytest
```
